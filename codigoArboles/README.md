El objetivo es analizar el rendimiento de los arboles ABB y B+, junto con el de una lista, despues de insertarlos en orden o en desorden.

* `ArbolABB.py`: Implementación del Árbol Binario de Búsqueda.
* `arbolBplus.py`: Implementación del Árbol B+ (nodos internos y hojas ligadas).
* `main.py`: Script de pruebas que genera los datos y mide los tiempos.

# Instrucciones
1. Clonar este repositorio o descarga los 3 archivos en una misma carpeta.
2. Abre una terminal o consola en dicha carpeta.
3. Ejecuta el script principal python main.py. Puede modificar las constantes para evaluar el rendimiento del programa, tales como:
*cantidad de registros a generar 
*cantidad de ids a buscar
*cantidad de veces a repetir para promediar
*el limite de claves-referencias por nodo en el arbol b+

    Una vez seteadas las constantes (ya llevan una configuracion por defecto), solo se debe correr ***main.py***.
